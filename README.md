# Homebrew Tap — ARO Applications

Homebrew formulae for applications built with the [ARO programming language](https://arolang.github.io/aro/).

## Usage

```bash
brew tap arolang/applications
```

### Available Formulae

| Formula | Description |
|---------|-------------|
| `sf` | Midnight Commander-style terminal file manager |

```bash
brew install arolang/applications/sf
```

## Updating

```bash
brew update
brew upgrade sf
```
